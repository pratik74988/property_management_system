# Property Management System

A full-stack property management system built with Django, supporting property listings, user authentication, personalized filtering, and administrative workflows.

## Features
- User authentication
- Property listings
- Preference-based filtering
- Admin dashboard
- Contact integrations (Call / WhatsApp / Email)

## Tech Stack
- Django
- PostgreSQL
- HTML/CSS/JavaScript

## Setup
```bash
git clone <https://github.com/pratik74988/property_management_system>
cd property-management-system
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
